⚠ Setup Required — Read Before First Use
▼ show

Ollama must be installed and configured to accept browser connections. Follow these steps once:

1
Download and install Ollama from ollama.com
2
Open a Command Prompt (Windows) or Terminal (Mac/Linux) and run this command to allow browser access:
setx OLLAMA_ORIGINS "*"
3
Reboot your computer. The environment variable does not take effect until after a full restart.
4
After rebooting, start Ollama (it runs in the system tray on Windows). Then return here and click the status pill at the top right to verify the connection.
5
Pull a model using the field below, or run ollama pull gemma4:12b in the terminal.
